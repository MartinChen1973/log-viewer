# 冰雪川（Bingchuan）冰淇淋产品知识库

## 品牌简介

冰雪川（Bingchuan）为本示例知识库中的冰淇淋产品线名称。以下条目供客服与内部检索使用。

## 经典口味

- **奶盐焦糖**：含乳制品与焦糖糖浆；含麸质车间可能交叉接触。
- **抹茶白巧**：含乳制品、大豆；咖啡因含量低。
- **莓果雪酪**：不含乳制品（雪芭系列）；含真实莓果泥。

## 过敏原与饮食提示

- 含乳制品的杯装与甜筒在配料表与杯身标签标注 **牛奶**。
- 坚果风味款（如榛子脆壳）在产线说明中标注 **树坚果**；门店可提供成分表复印件。
- 素食者可选择标注为雪芭（Sorbet）且不含蜂蜜的口味。

## 门店与零售政策（示例）

- 标准品脱装建议零售价以当季价签为准；促销堆头需保留原价对比标签至少 7 天。
- 融损或温控异常批次不得上架，应按 **冷链异常流程** 登记并报备区域督导。

## 常见问答要点

- **是否含人工色素**：经典系列以天然来源色素为主；具体以包装配料表为准。
- **儿童食用建议**：含咖啡因或高糖的口味包装上有相应提示；建议家长按年龄与体重适量给予。
